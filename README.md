### Lancement du projet

1)

- Clonez le projet dans votre éditeur de code :
`https://github.com/nathanremond/jikana.git`

2)

Pour lancer l'api, exécutez :

- cd api
- npm i
- npx nodemon index.js

3)

Pour lancer le site, ouvrez un second terminal et exécutez :

- cd frontend
- npm i
- npm run dev
